# ESP Get Pokemon API

Proyecto en el que combinamos el uso del modulo ESP8266 con el firmware NODE-MCU un OLED SSD1306 y la api de [pokeapi.co](http://pokeapi.co/).

## Material
- ESP8266-03
- Pantalla OLED SSD1306 128x64
- FTDI frend
- Protoboard
- Jumpers

## Libs
- cjson
